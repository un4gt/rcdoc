## 如何生成ApiKey？

1. 进入后台页面，点击左侧面板的 `令牌管理` 一栏

2. 如图，点击 `创建密钥` 按钮

3. 创建密钥的几个注意事项：

    - 最好不要开启 `可用模型限制`，保持默认即可
    - 如果你只想使用Codex套餐，不想让你的额度有额外流失，请关闭 `允许使用余额`

## Key 权限说明（很重要）

ApiKey 可以单独做权限控制，最常见就是这两项：

- `可用模型限制`：可以只放开一部分模型
- `允许使用余额`：决定套餐用完后，能不能继续走余额按量扣费

:::warning 常见问题提醒
如果你在调用时看到“模型不可用”“没有权限”“请求被拒绝”这类提示，先不要急着换模型，先检查当前 ApiKey 的权限配置。  
很多时候不是模型坏了，而是这个 Key 把模型限制住了，或者关闭了余额权限。
:::

### 快速排查顺序（建议）

1. 先看这个 Key 有没有开启 `可用模型限制`
2. 再确认你现在调用的模型是否在允许列表里
3. 如果套餐已用完，再看 `允许使用余额` 是否打开
4. 如果余额权限关闭或余额不足，可能会提示 `429`
